# Vyla Home

Vyla Home is a desktop version of Vyla, built to run entirely on your own computer instead of on [vyla.cc](https://vyla.cc).

## Why this exists

Running Vyla as a hosted website has been a constant source of stress. Hosting providers routinely block or throttle traffic coming from data centers, which is exactly where most affordable hosting lives. Every workaround for that involves paid infrastructure — residential proxies, dedicated IPs, premium tiers — that isn't realistic to keep funding long-term.

On top of that, there's the standing fear that comes with depending on someone else's infrastructure: a provider changes their policy, a server gets flagged, an account gets suspended, and everything just stops working with no warning. That uncertainty has genuinely cost some sleep. Waking up wondering if the whole project got wiped overnight isn't a sustainable way to build something.

Vyla Home is the answer to that. Instead of fighting hosting providers forever, the app runs locally on your own machine. No data center traffic to block. No hosting bills. No one else's finger on the switch.

## What it actually is

Vyla Home packages up the same player, streaming, and live-sports features as the website, wrapped in a small desktop app. When you open it:

- It grabs the latest version automatically
- It starts everything up locally on your computer
- The interface opens in its own window, just like the website did

Everything runs for you, on your machine, for your own personal use.

## What this is not

This isn't a public service or a way to redistribute anything. It's a personal media player for personal use — think of it as watching things you already have access to, through your own local player, instead of a public streaming platform. Nothing here is intended to be shared, resold, or run as a service for other people.

## Status

Might not be actively worked on. Expect various bugs — this is a solo project built to solve a real problem, not a polished commercial product.
